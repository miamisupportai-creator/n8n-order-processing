# E-commerce Order Automation

Processes inbound e-commerce orders via webhook, flags high-value orders over $500 with instant alerts, calculates fulfillment details, updates order status, and sends confirmation to the customer.

## Quick Start
1. Download `workflow.json`
2. Import into n8n: **Settings → Import Workflow**
3. Replace `${CLIENT_ID}` and `${CLIENT_NAME}` with your values
4. Configure credentials (Zoho CRM, Claude API)
5. Activate and test
